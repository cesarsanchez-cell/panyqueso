# Escudos de clubes (FUT-89)

Un PNG por club, nombre exacto **`<slug>.png`** (el slug es el `id` del catálogo
`lib/clubs.ts`). Se sirven en `/clubs/<slug>.png`. Si faltara alguno,
`<ClubCrest>` cae a un fallback con las iniciales.

- **Formato:** PNG · ~16–128 px (se muestra a ~16–22 px).
- Escudos = marcas registradas; uso para app privada.

Están los 51 clubes del catálogo. Para reemplazar uno, pisá el `<slug>.png`
correspondiente.
