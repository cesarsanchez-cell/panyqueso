# Escudos de clubes (FUT-89)

Un PNG por club, nombre exacto **`<slug>.png`** (el slug es el `id` del catálogo
`lib/clubs.ts`). Se sirven en `/clubs/<slug>.png`. Mientras falte un archivo,
`<ClubCrest>` muestra un fallback con las iniciales — se pueden ir sumando de a poco.

- **Formato:** PNG con fondo transparente · ~64–128 px.
- Origen de los actuales: imagen principal del artículo del club en Wikipedia
  (escudos = marcas registradas; uso para app privada).
