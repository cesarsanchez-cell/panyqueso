# Escudos de clubes (FUT-89)

Poné acá un PNG por club, con el nombre exacto **`<slug>.png`** (el slug es el `id`
del catálogo en `lib/clubs.ts`). Mientras falte un archivo, `<ClubCrest>` muestra
un fallback con las iniciales — así que podés ir cargándolos de a poco.

- **Formato:** PNG con fondo transparente.
- **Tamaño sugerido:** 64×64 o 128×128 px (se muestra a ~16–22 px).
- **Ruta final:** se sirven en `/clubs/<slug>.png`.
